# Project Mobile App

Flutter-based mobile application designed for field agents.
Supports offline data capture, visit tracking, and sync with the central backend.

## Tech Stack
- Flutter
- SQLite / Hive (offline storage)
- Google Maps SDK

## Features
- Offline-first data entry
- Visit and route tracking
- Background sync
- Authentication

## Setup
flutter pub get
flutter run
